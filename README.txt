inpPre-Requisite: nodeJS and monogo installed

Steps to start Application
==========================

1) Extract the zip file

2) Goto the extracted folder from commnad prompt

3) Run the below command

npm install

4) start the application with below command

node userapp.js

5) Access the below URL from browser

http://localhost:3000

6) To know the existing user details,please acccess the below url

http://localhost:3000/get
